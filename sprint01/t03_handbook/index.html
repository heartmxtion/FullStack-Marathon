HTML

Introduction to HTML

HTML is the standard markup language used to create web pages. It stands for Hypertext Markup Language.

<!DOCTYPE html>
<html>
<head>
	<title>My Web Page</title>
</head>
<body>
	<h1>Welcome to my Web Page</h1>
	<p>This is my first paragraph.</p>
</body>
</html>

HTML Elements

HTML elements are the building blocks of HTML pages. An HTML element is defined by a start tag, some content, and an end tag.

<p>This is a paragraph.</p>

<a href="https://www.example.com">This is a link</a>

<img src="image.jpg" alt="This is an image">

HTML Forms

HTML forms are used to collect data from users. The most common form element is the input element.

HTML Tables

HTML tables are used to display tabular data. The table element is used to create a table, and the tr element is used to create a row.

Name Age Gender
John 25 Male
Jane 30 Female

CSS

Introduction to CSS

CSS is the language used to style HTML pages. It stands for Cascading Style Sheets.

body { background-color: #f5f5f5; font-family: Arial, sans-serif; }

CSS Selectors

CSS selectors are used to select HTML elements based on their attributes, such as class, id, or tag name.

.button { background-color: #007bff; color: #fff; padding: 10px; border-radius: 5px; } #header { background-color: #fff; color: #000; } h1 { font-size: 2.5rem; } img { max-width: 100%; height: auto; }

CSS Box Model

The CSS box model is used to describe the layout and sizing of HTML elements. Each element has a content area, padding, border, and margin.

.box { width: 200px; height: 200px; padding: 20px; border: 1px solid #000; margin: 10px; }

CSS Flexbox

CSS flexbox is a layout model used to create flexible and responsive layouts. Flexbox is designed to work with one-dimensional layouts such as rows or columns.

.container { display: flex; flex-direction: row; justify-content: center; align-items: center; } .item { flex: 1; text-align: center; }

Web protocols

Introduction to Web Protocols

Web protocols are used to enable communication between web servers and clients. The most common web protocols are HTTP, HTTPS, DNS, and IP.

HTTP and HTTPS

HTTP is a protocol used for transmitting data over the web. HTTPS is a secure version of HTTP that uses encryption to protect data.

HTTP Methods:

  • GET - retrieves information from the server
  • POST - sends data to the server to create or update a resource
  • PUT - sends data to the server to update an existing resource
  • DELETE - sends a request to the server to delete a resource

HTTP Response Status Codes:

  • 1xx - informational
  • 2xx - successful
  • 3xx - redirection
  • 4xx - client errors
  • 5xx - server errors

DNS

DNS is a protocol used for translating domain names into IP addresses. DNS records include:

  • A - maps a domain name to an IPv4 address
  • AAAA - maps a domain name to an IPv6 address
  • CNAME - maps a domain name to another domain name
  • MX - maps a domain name to a mail exchange server
  • TXT - contains text information about a domain name

IP Addresses

IP addresses are unique identifiers assigned to devices connected to a network. There are two versions of IP addresses: IPv4 and IPv6. IPv4 addresses are 32-bit numbers, while IPv6 addresses are 128-bit numbers.

To determine the IP address of a domain name, you can use the nslookup command in the terminal. For example:

nslookup google.com

This will return the IP address of the Google domain.
